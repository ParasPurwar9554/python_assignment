# CALCULATOR USING TKINTER

A simple GUI-based calculator built using Python Tkinter.
This application allows basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division, with an interactive interface and clickable number buttons.

### Functionality
1. Clean and simple UI
2. Supports basic operations:
3. Addition (+)
4. Subtraction (-)
5. Multiplication (*)
6. Division (/)
7. Clear button to reset input
8. Uses Tkinter's Entry, Button, and event handling
9. Beginner-friendly Python GUI project

# How to Run
1. Save the script as calculator.py.
2. Run the file using: python calculator.py
